# Start Here

This repository is best used as a guided learning journey rather than a single project. The folders are a mix of classroom snapshots, runnable demos, and larger portfolio-style applications. This guide helps you move through them without getting lost.

## Who this repo is for

- Beginners who want a practical path from API calls to full AI applications
- Java and Spring developers moving into AI engineering
- Python developers exploring LangChain or LangGraph workflows
- Professionals comparing different ways to build RAG, tool-calling, MCP, and agent systems

## How to study each folder

For each lesson or project, try this loop:
1. Read that folder's `README.md` first
2. Identify the main concept and the runnable entry point
3. Run the smallest possible example
4. Change one input or prompt and observe behavior
5. Write down one takeaway before moving on

## Suggested weekly rhythm

- Week 1: SDKs, prompts, embeddings
- Week 2: vector stores, RAG, tool-calling
- Week 3: MCP, agents, LangGraph
- Week 4: end-to-end projects and comparisons

## Tips for professionals

- Focus on the comparison between Spring AI and LangChain approaches
- Treat the numbered folders as isolated implementation patterns
- Use the project folders to study packaging, architecture, and integration tradeoffs
- Pay special attention to setup notes, troubleshooting notes, and production considerations in the larger project READMEs
