OpenAI RAG (LangChain + Chroma)
--------------------------------------------------------------------------------------------------------------

Step 1: Create & activate virtual environment
  python -m venv .venv
  .venv\Scripts\activate   # Windows


Step 2: Install required libraries
  uv add langchain langchain-openai langchain-community langchain-chroma chromadb python-dotenv


Step 3: Add OpenAI API key : Create a .env file in project root
  OPENAI_API_KEY=your_openai_key_here

Gemini RAG (LangChain + Chroma)

Step 1: Create & activate virtual environment
  python -m venv .venv
  .venv\Scripts\activate   # Windows


Step 2: Install required libraries
  uv add langchain langchain-google-genai langchain-community langchain-chroma chromadb python-dotenv


Step 3: Add Google API key
  GOOGLE_API_KEY=your_gemini_api_key_here
